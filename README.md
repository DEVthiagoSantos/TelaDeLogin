# TelaDeLogin

Este projeto deixou de ser apenas uma tela de login e evoluiu para um sistema completo com integração a banco de dados.

Foi implementada uma conexão com MySQL utilizando JDBC para exibir e manipular dados em uma tabela, funcionando basicamente como um CRUD (Create, Read, Update e Delete).

A estrutura do sistema é genérica, podendo ser adaptada para diferentes cenários, como:
- cadastro de usuários
- controle de produtos (sistema de mercadinho)
- sistema de biblioteca (livros, empréstimos, devoluções e taxas)

## Tecnologias utilizadas

- JavaFX — construção da interface gráfica
- JDBC — conexão com banco de dados MySQL
- CSS — estilização da interface
